# HerosOfDestiny
## 예전에 자바로 만들었던 CLI 환경의 게임. 범위를 벗어나는 숫자나 입력에 올바르지 않은 문자등을 입력 했을시 예외사항처리에 애를 먹었던 기억이 난다.
## 여러 상황을 사용자입장에서 시뮬레이션 해보고 테스트를 많이 해봐서 오류를 최대한으로 줄였다. 이때에는 TDD라는 개발방식을 몰랐지만, 테스트가 중요하다 라는 것을 깨달았었다.
